A. 핵심 방법 논문 — "다 측정하지 않고" 혼합물 다루기

가장 직접적무료 전문 순물질만 학습 → 혼합물 인식

SERS Mixture Recognition from Pure-Substance Spectra via Component Evidence Learning and Two-Stage Inference · Molecules 2025 · 10.3390/molecules31091412

혼합물은 한 번도 학습 안 하고 순물질만으로 이진·삼진 혼합물 98.6% 인식. 구조: 1D-CNN/ResNet1D 인코더 + 성분별 evidence 분기(다중라벨) + 2단계 추론 (후보 선정 → 순물질 프로토타입으로 재구성 검증).

왜 중요: 내가 가진 순물질 DQ/THI/TBZ + 소수 Ratio로 바로 적용 가능. 키워드 — 1D-CNN, multi-label, evidential learning, template reconstruction.

본문 유료 · 데이터 공개미측정 분자의 스펙트럼까지 예측

ML-Based SERS Chemical Space for Two-Way Prediction of Structures and Spectra of Untrained Molecules · JACS 2024 · 10.1021/jacs.4c15804 · PubMed 39950227 · figshare 데이터

학습 안 한 분자의 구조↔스펙트럼 양방향 예측(코사인 90.4%). 방법: XGBoost + 단순 descriptor(작용기 + 탄소 사슬 길이), 딥러닝 아님. 데이터 38개 선형분자·4계열.

왜 중요: 라이브러리를 계산으로 확장하는 접근. SI로 재현 가능.

B. 경쟁흡착·친화도(K) — 왜 신호≠농도인가, 어떻게 보정하나

DFT로 SERS를 어디까지 예측하나 (한계 명시)

Can DFT Calculations Provide Useful Information for SERS Applications? · PMC9861783

DFT는 피크 위치·진동모드 할당·흡착 기하·상대세기는 O, 절대 강도·핫스팟·전자기 증강은 X. 해석 보조지 예측 엔진 아님. → 경쟁흡착의 "친화도 K" 계산엔 맞고, "세기 예측"엔 안 맞음.

흡착에너지(=친화도)를 빠르게 — ML 원자간 퍼텐셜

Pretrained MLIP in Heterogeneous Catalysis (PMC12976938) · CatBench (Cell Rep. Phys. Sci.)

풀 DFT 없이 사전학습 모델로 K_i(상대 친화도)를 스캔 → 경쟁흡착 forward 모델 파라미터화. (Open Catalyst 계열)

D. (초기 논의) 후각·구조→지각 매핑 — 배경

Principal Odor Map (Osmo)

A Principal Odor Map Unifies Diverse Tasks in Olfactory Perception · Science 2023 · PMC11898014 · 데이터: Pyrfume, DREAM Olfaction Challenge

구조→냄새 GNN 예측. "구조→성질 예측"이라는 같은 틀(발상의 뿌리).

방법·개념 체크리스트
  • 비율 복원: 순물질 템플릿에 NNLS → C_i:C_j = (B_i/R_i):(B_j/R_j). 기판 gain·경쟁항이 비율에서 상쇄 → 재현성 우회.
  • 응답계수 R_i = A_i·K_i: 알려진 혼합물 1개로 보정(순물질 곡선만으론 경쟁 못 담음).
  • 절대농도(µM): 내부표준으로 gain·포화까지 상쇄(r ∝ C). 단 IS가 분석물과 선형독립이어야(겹침 = collinearity 확인).
  • 두 물리 한계: 표면 포화(지배종 농도 소실→희석) / 노이즈 바닥(묻힌 성분<LOQ → SNR·마커밴드·표준첨가).
  • 내부표준 겹침 대응: 단일밴드 대신 전체 스펙트럼 동시 피팅(IS도 성분으로).
내 데이터로 확인된 것 (Pest_Discriminator)
  • 순물질 DQ/THI/TBZ 조건수 ≈ 2.5 → 잘 분리됨(전체 스펙트럼 unmixing 유효).
  • ~2132 cm⁻¹ 피크: 블랭크엔 거의 없고 THI/TBZ에서 강함 → 기판 고유 아님, 질소 분석물의 표면 광생성물(Ag–CN/무정형 탄소) 의심 → 내부표준 부적합.
