DQ / THI / TBZ 농약 — 순물질로 혼합물 성분·비율·농도 뽑기 · 경쟁흡착 · 기판 재현성

있음보유 데이터 (Pest_Discriminator)
  • 순물질 레퍼런스 — DQ, THI, TBZ, 블랭크(blk). 각 400픽셀, baseline 보정본.
  • 이진 혼합물 — 세 조합 전부 1:1 / 1:3 / 3:1: DQ+THI, TBZ+THI, TBZ+DQ.
  • 삼진 — 1:1:1 (THI1) 하나.
  • THI 농도 계열 — THI01(1:1:0.1), THI001(1:1:0.01). "묻힌 성분" 케이스 → 현재 학습 제외.
  • 타깃 — acf(깨끗한 기판, 실측 몰비 THI:TBZ:DQ = 0.1:1:1, 유일한 ground truth), 바나나·잎(food matrix).
  • 포맷 — XY맵 CSV, 100–3200 cm⁻¹, 2001 pts, 조건당 400 px(20×20).

핵심 결론 & 물리 한계
  • 비율 복원: 순물질 NNLS → C_i:C_j = (B_i/R_i):(B_j/R_j). gain·경쟁항이 비율에서 상쇄 → 재현성 우회.
  • 절대 µM: 내부표준으로 gain·포화까지 상쇄(r ∝ C). 단 IS가 분석물과 선형독립이어야.
  • 표면 포화: 지배종 농도 소실 → 희석으로 탈출.
  • 노이즈 바닥: 묻힌 성분 < LOQ → SNR·마커밴드·표준첨가.
